### ▶️ Inicialização

Inicialize o projeto com npm

```bash
  npm i -g @nestjs/cli
  cd backend && npm install
  npm run start
```

O banco presente nos arquivos já possoui algumas cadeiras cadastradas como exemplo, sendo necessario a importação para o phpMyAdmin ou outra aplicação.
